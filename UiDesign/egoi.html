Web Widget

Faculty of Fine Arts of University of Porto
Curricular Intership LGP FEUP
Project: UI Design for a Web Widget
Client: E-goi
2021 UI Design — Prototype — Figma
Interface visible to the end-customer
What is e-goi? E-goi is a multichannel marketing automation platform that sends thousands of campaigns each day through various communication channels, such as e-mail, SMS, Web Push.... Widget purpose Taking into account the current panorama, where companies are reinforcing their digital presence valuing customer proximity, it becomes essencial to extend the horizons of the already existing voice communication channels in order to improve the company's connection with the customer.
Widget Configuration Interface. Visible to e-goi's clients, for fully customization (from the text visible to the end-customer, to the visual aspects) of the widget to mach their company's identity.
