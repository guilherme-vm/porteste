Medove

Faculty of Fine Arts of University of Porto
Web Design II
Project: Tracker App
2020/2021 HTML/CSS — JavaScript — jQuery — Mobile/Desktop
Medove is your app for daily use. If you feel like you should start taking time for yourself each day, this is it.
Inside the app you can select a soundtrack, on the Music page, for the desired time selected on the Timer page. Within the Main page you can select your mood for the day with a range of smiles, and also create new notes to keep track of your daily reflections.
